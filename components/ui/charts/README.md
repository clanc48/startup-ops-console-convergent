This folder is kept for lightweight, dependency-free chart components used by optional pages.

## Current state
- The only chart component currently in use is `components/ui/MultiLineChart.tsx`.
- `MiniAreaChart` and `Sparkline` were removed as unused.

If you add more charts again, either:
- move them into this folder and update imports accordingly, or
- keep them in `components/ui/` and treat this folder as documentation-only.
